# Rails Engine

Th goal of this project is to expose data through an API for a company developing an E-Commerce Application.

- Ruby version 2.7.2

- ## System dependencies

  - rspec-rails
  - pry
  - shoulda-matchers
  - factory_bot_rails
  - faker
  - fast_jsonapi
  - will_paginate

- How to run the test suite
  - rspec
  - postman test suite
